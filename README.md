# Personal Portfolio

This is my personal portfolio, built from scratch.

## Run Locally

Clone the project

```bash
  git clone https://github.com/VEN-github/my-portfolio.git
```

Go to the project directory

```bash
  cd my-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Build for production

```bash
  npm run build
```
